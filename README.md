# FootZone

BTU Frontend Web Development - Final Project

A simple multi-page football website (inspired by LiveScore / ESPN).

## Pages

- **index.html** - home page with latest Premier League results (from API) and popular leagues
- **teams.html** - teams from 6 leagues fetched from TheSportsDB API, with league filter buttons, search and favorites
- **contact.html** - contact form with validation

## Used in this project

- HTML, CSS, JavaScript (3 pages)
- Flexbox (navbar, match cards, filter buttons) and Grid (league and team cards)
- Responsive design with @media queries
- DOM manipulation and event handling
- Fetch API with async/await (TheSportsDB)
- Form validation with JavaScript
- LocalStorage (favorite teams + saved messages)

## How to run

Just open `index.html` in a browser, or visit the live demo on GitHub Pages.
